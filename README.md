# Cookify Demo

Интерактивное демо-приложение для поиска рецептов и планирования питания.

## Установка и запуск

### Шаг 1: Установите зависимости
```bash
npm install
```

### Шаг 2: Запустите приложение
```bash
npm start
```

Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)

## Функциональность

- **Поиск рецептов** по названию или ингредиентам
- **Фильтры**: тип блюда, диета, время приготовления, кухня мира
- **Личный кабинет** с профилем и планом питания
- **Двуязычность**: русский и английский
- **Подсветка аллергенов** в ингредиентах

## Технологии

- React 18
- Tailwind CSS
- React Icons
